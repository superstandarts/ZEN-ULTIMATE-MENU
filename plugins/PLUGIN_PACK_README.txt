ZEN PLUGIN PACK
===============

Plugins included:

1. ZEN Wallpaper Changer
2. ZEN Roblox Tools
3. ZEN Steam Tools
4. ZEN Browser Cleaner
5. ZEN Vercel Helper
6. ZEN Badge Generator
7. ZEN Portfolio Builder
8. ZEN Clip Compressor
9. ZEN Audio Converter
10. ZEN Banner Maker
11. ZEN Sound Pack Installer
12. ZEN App Skin Manager
13. ZEN Project Scanner
14. ZEN Auto Updater Plugin
15. ZEN Plugin Creator

How to use:
- Open ZEN ULTIMATE MENU
- Go to Plugins
- Click List Plugins
- Type part of the plugin name
- Click Run Plugin

Notes:
- FFmpeg is required for Clip Compressor and Audio Converter.
- Vercel CLI is required for Vercel Helper.
- Skin Manager replaces files in assets/.
- Auto Updater currently creates backups; future versions can download releases.
