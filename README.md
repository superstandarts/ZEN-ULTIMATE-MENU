# ZEN ULTIMATE MENU ULTRA FIXED

This version improves dashboard scrolling, theme loading, music lyrics, loading screen image scaling and config usability.

## Fixes

- Scrollable dashboard
- Scrollable left menu
- All sidebar modules also appear on Dashboard
- Spotify music widget auto-updates
- Synced lyrics appear in the terminal
- Current lyric line is highlighted
- ZEN loading image keeps its aspect ratio
- ZEN image has gray background
- Theme presets now apply after restart
- config.json has `_quick_settings` at the top

## Run

```bash
pip install -r requirements.txt
python main.py
```

## Build

```bash
build_exe.bat
```

Keep `config.json` and `assets/` next to the `.exe`.
